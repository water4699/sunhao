# 阶段 C：AI 智能选课助手 — 答辩演示要点

## 功能位置

- 小程序首页 → **「智能选课助手」** → `pages/aiAssistant/aiAssistant.vue`
- 后端接口：`POST /system/ai/assistant`（**需登录**），实现模块：`ruoyi-ai`

## 演示脚本（约 2 分钟）

1. 使用**家长角色**账号登录（需已执行 `docs/sql/register_roles_parent_teacher.sql` 并完成注册绑定）。
2. 打开「智能选课助手」，选择地区 / 年级 / 科目（可选），在文本框输入例如：「孩子初二数学基础弱，怎么选老师？」
3. 点击 **获取建议**：
   - **已配置** `RUOYI_AI_API_KEY`：可见自然语言建议 + 下方推荐教师列表；界面 meta 含「已调用大模型」。
   - **未配置密钥**：仍展示说明文案 + 推荐教师；meta 为「规则回退 · 来源: no_api_key」，体现**失败回退不断线**。
4. 点击某位推荐教师，跳转教师详情，与「找老师」主流程衔接。

## 论文/说明书可写表述

- 大模型为**可选增强**；核心推荐逻辑为数据库规则（`status=1` 已审核教师 + 筛选条件 + 分页取前若干条）。
- 外部 API 超时或失败时自动回退到规则推荐，保证演示可用性。
- 密钥通过**环境变量**注入，不进入版本库。

## 录屏建议

- 同时准备「有 Key」与「无 Key」各录一段，或一段内先展示无 Key 再说明配置 Key 后的差异。
